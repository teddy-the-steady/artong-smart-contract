# Frequently used commands

## Init
```
npx hardhat
```

### Start local hardhat network
```
npx hardhat node
```

### Compile contracts
```
npx hardhat compile
```

### Run deployment script
```
npx hardhat run --network hardhat scripts/deploy.js
```

### Open hardhat console
```
npx hardhat console --network localhost
```

### Run tests
```
npx hardhat test
```
